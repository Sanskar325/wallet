## ğŸš€ Quick Start from GitHub

### Clone Repository
```bash
git clone https://github.com/Sanskar325/wallet.git
cd wallet
```

### Install Dependencies
```bash
# Backend
cd backend && npm install && cd ..

# dApp Web
cd dapp-web && npm install && cd ..

# Wallet Web
cd wallet-web && npm install && cd ..

# Wallet Native (optional)
cd wallet-native && npm install && cd ..
```

### Start Services
```bash
# Terminal 1 - Backend
cd backend && npm start

# Terminal 2 - dApp
cd dapp-web && npm run dev

# Terminal 3 - Wallet
cd wallet-web && npm run dev
```

### Test It!
1. Open http://localhost:3000 (dApp)
2. Open http://localhost:3001 (Wallet)
3. Click "Scan & Connect"
4. Scan QR code!
