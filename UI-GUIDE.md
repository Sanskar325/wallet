- Elevated cards

## Design Principles

### 1. Minimalism
- Clean layouts
- Ample whitespace
- Focus on essential elements
- No clutter

### 2. Premium Feel
- Dark theme
- Neon accents
- Glowing effects
- Smooth animations

### 3. Clarity
- Clear hierarchy
- Readable text
- Obvious actions
- Instant feedback

### 4. Consistency
- Same colors throughout
- Consistent spacing
- Unified typography
- Predictable behavior
